Sensotrend's IPS Implementation

This implementation guide describes Sensotrend's implementation of the HL7 FHIR IPS format of the International Patient Summary (IPS) specification both as a client app and as a FHIR server. It also includes and extensive example patient summary for Mikael, an example patient.

This is an R4 IG. None of the features it uses are changed in R4B, so it can be used as is with R4B systems. Packages for both R4 (fi.mikaelrinnetmaki.ips.r4) and R4B (fi.mikaelrinnetmaki.ips.r4b) are available.
